About

Deepemo.tech is a facial recognition tool that takes an image or live camera feed and displays the position of faces and their emotion, along with a confidence rating for each. This confidence rating can range from 0.01 to 0.99, and indicates how confident the algorithm was in its result. Positions of faces and their ratings are marked with a blue box, while the emotion and their rating is below the blue box. This application consists of 6 pages:

Home

The default screen. Information is located at the top right and the detection application is available when you scroll down.

User Manual

The current page, which intends to act as a guide for the website.

Privacy

A page that describes our websites privacy policy.

Profile Choice

This page allows you to navigate to the live detection page by choosing one of three performance profiles. The Performance Profile is designed for mobile and runs the fastest, the Balance Profile is the recommended choice for any desktop, and the Greek Profile is designed for accuracy.

Image Detection

This page allows you to submit an image to be run by the algorithm. After allowing access to choose an image, the result will be returned to the screen. This can be saved by right-clicking ‘Save as’.

Live Detection

This page, once allowed access to a camera, will continuously return a live detection video. There are two updating counters that report your current frames per second and latency to the server
